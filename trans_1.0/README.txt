说明：
运行dist文件夹下的trans_version_1.0.exe

在控制台输入要查询的单词

会根据你所输入的单词生成一个名为该单词的txt文件

txt文件里包含该网站查询到的相关信息，包括音标，词义和例句，德语专业词典，德语例句库。

若某条信息不存在，输出为NONE



注意：
1.未加隐藏。不要一个ip某个时间段一直查同一个单词，他会返回图片格式的汉字，由于这类图片出现位置不固定，所以没法针对这一问题。
该问题主要出在汉字上，不行就自己“手工智能”吧。不过可以加个TensorFlow，emmm有能力的大佬整吧。

2.估计还有不少我没测出来的bug，代码很乱，很多地方不规范，正则与bs4混用，时间优化等问题以后会优化。

3.德语例句库中某些句子包含一个以上<span class="key">“关键字”</span>的，就不输出了，处理起来麻烦。

4.该网站还有其他各种词典，以后会添加查询其他词典的功能。
